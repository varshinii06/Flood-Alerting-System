# IoT Based Flood Monitoring System

This project is an IoT-based flood monitoring and alert system using Arduino and ESP8266.

## Features
- Real-time water level monitoring
- Ultrasonic sensor based detection
- LED and buzzer alerts
- LCD display output
- Telegram notification system

## Components Used
- Arduino UNO
- ESP8266 WiFi Module
- Ultrasonic Sensor
- LCD (I2C)
- Buzzer
- LEDs
- Potentiometer

## Working
The ultrasonic sensor measures water level. Based on distance:
- Safe → No alert
- Warning → LED indication
- Danger → Buzzer + Telegram alert

## Applications
- Flood monitoring
- Dam water level monitoring
- Disaster management
